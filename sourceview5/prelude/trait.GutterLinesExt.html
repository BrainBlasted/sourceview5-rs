pub fn add_class(&self, line: u32, name: &str)[src]

Adds the class name to line.

name will be converted to a glib::Quark as part of this process. A faster version of this function is available via GutterLinesExt::add_qclass for situations where the glib::Quark is known ahead of time.

Feature: v5_0

line

name

pub fn add_qclass(&self, line: u32, qname: Quark)[src]

Adds the class denoted by qname to line.

You may check if a line has qname by calling GutterLinesExt::has_qclass.

You can remove qname by calling GutterLinesExt::remove_qclass.
